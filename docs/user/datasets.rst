IMAGE_CLASSES_BY_FOLDERS
--------------------------------------------------------------------

Protocol of URI must be ``http`` or ``https``.

The dataset file must be of the ``.zip`` archive format with the following folder structure:

.. code-block:: bash

    class_1/
        image_1.png
        image_2.png
        image_3.png
    class_2/
        image_4.png
        image_5.png
        image_6.png
    ...
    class_n/
        image_k-2.png
        image_k-1.png
        image_k.png
    
Supported Image File Formats
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
- ``.png``
- ``.jpg``
- ``.jpeg``
